# npm 插件

快捷的查询 npm 库（npm helper）

![npm helper 插件](../demo/2020-06-18_173702.jpg)

使用了 vue + vant + typescript 进行项目开发


# !!!注意⚠️!!!
部分数据收藏在了自己的服务器（腾讯云开发）中，所以请修改源代码之后再使用该插件
